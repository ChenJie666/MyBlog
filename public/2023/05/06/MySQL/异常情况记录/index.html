###1. 创建唯一主键报错
现象：创建唯一主键报错时报错：Specified key was too long; max key length is xxx bytes
原因：因为该主键为varchar型，长度为255。长度过长报错。
措施：将长度改为191以下。
JPA创建表时注意添加注解 @Column(length = 191,unique = true)
